
<h1>QR Code Generator</h1>
<hr>
Python app with tkinter to generate qr code of given text and save it in .png format.
<hr>

<h3>Modules used →</h3>
▶ qrcode<br>
▶ Tkinter
<hr/>
<h3>How to run →</h3>
● Install module named 'qrcode'<br><br>
<pre>pip install qrcode</pre><br>
● run the file qr.py<hr>
